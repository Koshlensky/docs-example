# docs-example

Пример репозитория с документацией, оформленной в **AsciiDoc**.

В репозитории собраны примеры двух типов документации:

- **API documentation** — находится в папке `api-description`
- **User guides** — находятся в папке `user-guide`

Также в репозитории есть общие ресурсы для оформления и просмотра документации:

- `styles` — стили для HTML-страниц
- `images` — изображения, используемые в документации

## Структура репозитория

```text
docs-example/
├── api-description/
│   ├── api-create-method.adoc
│   ├── api-service-example.adoc
│   ├── api-create-method.html
│   └── api-service-example.html
├── user-guide/
│   ├── calculator-user-guide.adoc
│   ├── whatsapp-broadcasting-user-guide.adoc
│   ├── calculator-user-guide.html
│   └── whatsapp-broadcasting-user-guide.html
├── images/
├── styles/
├── .gitignore
└── README.md
```

## Описание содержимого

### `api-description/`

Папка с примерами документации по API.

Здесь находятся:

- файлы `.adoc` — исходники документации в формате **AsciiDoc**
- файлы `.html` — готовые сгенерированные страницы, которые можно открыть в браузере для просмотра

### `user-guide/`

Папка с примерами пользовательских руководств.

Здесь также находятся:

- файлы `.adoc` — исходные тексты руководств в формате **AsciiDoc**
- файлы `.html` — готовые HTML-файлы для просмотра

### `styles/`

Папка со стилями для HTML-страниц документации.

### `images/`

Папка с изображениями, используемыми в документации.

## Форматы файлов

- `.adoc` — исходные файлы документации в формате **AsciiDoc**
- `.html` — уже готовые файлы документации, открываемые в браузере

## Назначение репозитория

Этот репозиторий можно использовать как пример организации проекта с документацией, где:

- исходники хранятся в формате **AsciiDoc**
- рядом лежат уже собранные **HTML-файлы**
- общие изображения и стили вынесены в отдельные папки

## Просмотр документации

Для быстрого просмотра можно просто открыть нужный файл `.html` в браузере.

Например:

- `api-description/api-create-method.html`
- `user-guide/calculator-user-guide.html`
