# Employee README

В этом репозитории собраны материалы по работе с внутренним сервисом сотрудников **Employee**.

- `working-with-employee.adoc` — исходная статья в AsciiDoc. Формат удобен для редактирования и конвертации в HTML, PDF, Confluence и другие Wiki-системы.
- `working-with-employee.html` — сгенерированная HTML-версия статьи для быстрой демонстрации содержания.
- `images/` — директория с изображениями, используемыми в документации. Переименование или перенос этой директории потребует обновить пути к изображениям в `working-with-employee.adoc`.
- `questions.md` — список вопросов к разработчику по поведению сервиса Employee.

Для интеграции в свою инфраструктуру достаточно использовать `.adoc` как первоисточник и конвертировать его в нужный формат в любом IDE.
